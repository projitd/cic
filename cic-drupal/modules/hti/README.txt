Hierarchical Taxonomy Import
============================
This module imports taxonomy term on parent-child relationships from a CSV file.

Known Bugs
==========
It doesn't updates the existing taxonomy tree. So please be extra cautious while
importing the data and validate it before starting import.

Steps to Import
===============
1. Install the module.
2. Go to Admin/Configurations and then click on Taxonomy Importer.
3. Form will appear where you would select vocabulary to import the data.
4. Upload your CSV file and click on Submit button.
